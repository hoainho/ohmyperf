OhMyPerf

Real-machine, real-browser web performance measurement with ~99% cross-origin iframe coverage. Runs on your hardware, not a synthetic datacenter CPU.

Install the CLI Drop a report here Get the Chrome extension GitHub

Why?

Lighthouse and PageSpeed Insights run on synthetic emulated CPUs in Google's datacenter. The numbers are systematically inflated and don't reflect what your real users experience on real hardware. Existing tools also can't deeply inspect cross-origin iframes — the long-standing blind spot of web-perf tooling.

What you get

  • Real numbers from real browsers — Chromium runs on your machine; no synthetic CPU throttle by default.
  • Cross-origin iframe deep-inspection via CDP Target.setAutoAttach (~99% measurable signals).
  • Plugin-first — every metric, audit, reporter is a plugin. web-vitals, axe-core, custom metrics.
  • Honest variance — every report carries CoV; unstable runs (CoV>20%) get a visible banner.
  • CI-Stable mode with pre-flight CPU calibration so numbers compare across runners.
  • Inert shared reports — public viewers never re-execute plugin code.

Install the CLI

npm install -g @ohmyperf/cli
ohmyperf install-browser
ohmyperf run https://example.com --runs 5 --format json,html

Requires Node.js 20+ on macOS, Linux, or Windows.

Chrome extension

Install the extension, click the toolbar icon on any tab to measure it. The result opens in a new tab with the full HTML viewer.

Available for Chrome and Edge. Firefox and Safari support uses the CLI instead (CDP isn't available in those browsers).

Drop an existing report

Already have a report.json? Open the drag-drop viewer — no upload, runs entirely in your browser, zero network requests.
